# HDRV — Konohagakure Hub

Plataforma de comunidad para Naruto Online con sistema de guías, XP, logros y rankings.

## Tech Stack

- **Frontend:** Next.js 16 + React 19 + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **Base de datos:** PostgreSQL via Neon.tech (Prisma ORM) / SQLite en desarrollo
- **Auth:** JWT + bcrypt
- **Hosting:** Netlify (frontend) / Render.com (backend)

## Desarrollo local

```bash
npm install
npm run dev
```

Frontend: http://localhost:3000  
Backend: http://localhost:4000
